현재 대화에서 작업한 내용을 기반으로 코드와 문서가 일치하는지 확인하고 동기화해줘.

## 실행 단계

1. **대화 맥락 파악**
   이 세션에서 뭘 했는지 대화 내용을 보고 파악:
   - 어떤 파일을 수정했는지
   - 어떤 기능을 추가/변경/삭제했는지
   - 어떤 설정을 바꿨는지

2. **관련 문서 확인**
   변경된 내용과 관련된 문서 찾기:
   - README.md
   - 해당 기능 관련 문서

3. **코드-문서 일치 확인**
   - 변경된 코드가 문서에 반영되어 있는지
   - 문서 내용이 현재 코드와 맞는지

4. **필요시 문서 업데이트**
   불일치하는 부분 수정

5. **결과 보고**
   뭘 확인하고 뭘 바꿨는지 간단히 알려줘.

---

git 전체를 보고 꼼꼼하게 동기화하려면 `/sync-full` 사용.
