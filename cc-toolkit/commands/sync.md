방금 이 세션에서 수정한 내용을 기반으로 코드와 문서가 일치하는지 꼼꼼하게 확인하고 동기화해줘.

## 실행 단계

1. **방금 수정한 내용 파악**
   이 대화에서 뭘 수정했는지 맥락을 보고 파악:
   - 어떤 파일을 수정/추가/삭제했는지
   - 어떤 기능을 변경했는지
   - 어떤 설정을 바꿨는지

2. **관련 문서 찾기 및 읽기**
   수정한 내용과 관련된 문서를 찾아서 읽어:
   - README.md
   - 해당 기능 관련 문서
   - plugin.json, package.json 등 메타 파일

3. **코드-문서 불일치 체크**
   꼼꼼하게 확인:
   - [ ] 수정한 기능이 문서에 반영되어 있는가?
   - [ ] 문서 내용이 현재 코드와 맞는가?
   - [ ] 버전 정보가 일치하는가?
   - [ ] 예제/사용법이 현재 코드와 맞는가?
   - [ ] 삭제된 기능이 문서에서도 제거되었는가?

4. **문서 업데이트**
   불일치하는 부분 수정:
   - README 업데이트
   - 관련 문서 수정
   - 버전 정보 동기화

5. **결과 보고**
   뭘 확인하고 뭘 바꿨는지 보고.
